Callback Hooks

▼
on_create_move(cmd, local_player)
Called every tick on the CreateMove hook. Use for aimbot, movement, and input manipulation.
cmd UserCmd User command object for current tick
local_player CTerrorPlayer Local player entity
Example:
function on_create_move(cmd, local_player) local angles = cmd:get_viewangles() angles.y = angles.y + 1.0 cmd:set_viewangles(angles) end
on_end_scene()
Called every frame on the rendering thread. Use for ImGui menus and rendering.
Example:
function on_end_scene() if imgui.begin("My Window") then imgui.text("Hello World") imgui.end() end end
on_paint()
Called every frame for ESP and 2D rendering.
Example:
function on_paint() draw.string(fonts.ESP, 100, 100, 1, 1, 1, 1, text_align.LEFT, "ESP Text") end
on_draw_model_execute(model_info)
Called before each model is rendered. Use for chams and model modifications.
model_info table Contains entity_index and model_name
Example:
function on_draw_model_execute(info) if info.entity_index > 0 then local mat = materials.find("vgui/white", nil) if mat ~= 0 then render.override_material(mat, 1, 0, 0, 1, false) end end end
on_game_event(event_name, event)
Called when a registered game event fires.
event_name string Name of the event
event IGameEvent Event object
Example:
events.register("player_death") function on_game_event(name, event) if name == "player_death" then local victim = event:get_int("userid") client.print("Player died: " .. victim) end end
on_cl_move_pre()
Called before CL_Move processes commands.
on_cl_move_post()
Called after CL_Move processes commands.
on_unload()
Called when script is being unloaded. Use for cleanup.
Example:
function on_unload() client.print("Script unloading, cleaning up...") end

Anti-AFK
local last_move_time = 0 function on_create_move(cmd, local_player) local current_time = client.get_tick_count() if current_time - last_move_time > 600 then local buttons = cmd:get_buttons() cmd:set_buttons(buttons | buttons.IN_JUMP) last_move_time = current_time end if cmd:get_forwardmove() ~= 0 or cmd:get_sidemove() ~= 0 then last_move_time = current_time end end
